# Liuyunbin-mh.github.io

A single-page personal blog and portfolio with a modern dark theme.

## Features

- **Single-page design** — Everything on one scrollable page
- **Fixed sidebar** — Profile info always visible while scrolling
- **Smooth navigation** — Click nav links to jump to sections
- **Active section highlight** — Navigation shows current section
- **Responsive** — Works on desktop, tablet, and mobile

## Structure

```
index.html              # Main single-page site
posts/                  # Blog post articles
  └── hello-world.html
assets/
  ├── css/style.css     # Global styles
  └── images/avatar.jpg # Your profile photo
```

## Sections

- **About** — Personal introduction
- **Education** — Academic background
- **Experience** — Work history
- **Publications** — Research papers
- **Skills** — Technical skills
- **Blog** — Latest posts

## Customize

1. Edit `index.html` and replace all placeholder text with your info
2. Replace `assets/images/avatar.jpg` with your photo (400×400px recommended)
3. Duplicate `posts/hello-world.html` to create new blog posts
4. Update the Blog section in `index.html` to link to new posts

## Deploy

Push to a GitHub repository named `<your-username>.github.io` and enable GitHub Pages.
